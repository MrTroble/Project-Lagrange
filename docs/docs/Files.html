ResourceDescriptor
 Holds the offsets and sizes of a resource in a resource file
  • offset is the offset in the file of the resource
  • size
    • is the size in byte of the resource

Valid usage

  • offset must be in the range from 0 to the size of the file
  • size
    • must be in the range from 1 to the filesize - offset 
    uint64_t offset;
uint64_t size;
