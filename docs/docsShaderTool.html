The Shader Tool

Note: The ShaderTool is currently under development and not yet finished

The Shader Tool is a dotnet tool written in C#. Unlike the previous tools this one is more likely a unix pipeline tool. So your are able to pipe commands in it and use the minimal output or the return codes for tool chaining. It has a command interface for starting it without parameters. But currently there is no GUI for the tool. This might change in the future.

Commands

Every command can have the command option --dir which sets the CWD of the according command. The value of the dir option can either be a path to a valid folder or "tool" which sets the CWD to the according folder if you start it from the IDE. If the option is not present it will take the normal CWD

pipe commands

Used to manage shader pipelines

pipe list

Lists all created pipes.

pipe create <name> <shader> [moreshader...]

Creates a new Shader pipe with the given name and shader.

pipe show <name>

Shows details to the given pipe.

pipe make

Generates the C++ code for all created pipes.

pipe update <name>

Updates the PipelineInputAttributes.

pipe delete <name>

Deltetes the given pipeline.
