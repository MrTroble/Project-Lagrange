Build on your own

To compile your own build you have to install the dependencies. To do so you need tu run python setup.py. There you have to select 1 Get dependencies Now you can open the VisualStudios Project file and compile the Engine.

Libraries

I am using LunarG's Vulkan-SDK.
Current built version: 1.1.101
Current Vulkan API version: 1.1

and Autodesk FBX SDK Current built version: 2019.2

For image loading and font loading is STB currently being used. This is planned to be replaced in the future. SIDE NOTE: The file library TGVL maybe needed in the future to compile newer versions of the TGEngine but currently it is not supported!
