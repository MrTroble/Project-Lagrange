Build on your own

You have to install the dependencies to compile your own build. You need to run python setup.py to do so. There, you have to select 1 Get dependencies. Now you can open the Visual Studio project file and compile the Engine.

Libraries

I am using LunarG's Vulkan-SDK.
Current build version: 1.1.101
Current Vulkan API version: 1.1

Autodesk: FBX SDK Current build version: 2019.2

STB is currently being used for image and font loading . This is going to be replaced in the future.

SIDE NOTE: The file library TGVL may be needed in the future to compile newer versions of the TGEngine, but currently it is not supported!
