Libraries

I am using LunarG's Vulkan-SDK.
Current built version: 1.1.82.0
Current Vulkan API version: 1.1

For image loading and font loading is STB currently being used.
This is planned to be replaced in the future. The file library TGVL is needed to compile newer versions of the TGEngine

NOTE: GLFW3 was removed because of linker errors
